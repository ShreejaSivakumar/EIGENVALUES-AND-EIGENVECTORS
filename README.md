# EIGENVALUES-AND-EIGENVECTORS
## Aim:
To write a python program to find the Eigenvalues and Eigen Vectors
## Equipment’s required:
1. 	Hardware – PCs
2. 	Anaconda – Python 3.7 Installation / Moodle-Code Runner
## Algorithm:

### Step 1: Start the program.

### Step 2: Import the NumPy library to perform matrix operations.

### Step 3: Define or input the matrix whose eigenvalues and eigenvectors need to be calculated.

### Step 4: Use the function np.linalg.eig() to compute both eigenvalues and eigenvectors of the matrix.

### Step 5: Display the eigenvalues and eigenvectors as output.

### Step 6: End the program.

## Program:

```PYHTON
#Program to find the eigen values and eigen vectors.
#Developed by: SHREEJA R S
#RegisterNumber: 25017561

import numpy as np
A=np.array([[2,-3,0],[2,-5,0],[0,0,3]])
values,vectors=np.linalg.eig(A)
print("Eigen values are {} and Eigen Vectors are {}".format(values,vectors))
```
